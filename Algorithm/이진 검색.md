이진 검색
===========

> 1) 선형 검색
--------------
- 시퀀스 전체 원소를 방문하면서 해당 원소가 N과 같은지를 확인하는 것

```C++
bool linear_search(int N, std::vector<int>& sequence)
{
    for(auto i : sequence)
    {
        if(i==N)
            return true;

    }

    return false;
}
```

- 입력 시퀀스의 정렬여부와 상관없이 항상 잘 작동, 시간 복잡도는 O(N)
- 그러나 효율적이지 않으며 정렬되어 있다면 전혀 이용하지 못함
